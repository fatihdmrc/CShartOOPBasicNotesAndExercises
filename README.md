
# Nesne Tabanlı Programlama Notları 

Bu repo, Bandırma Onyedi Eylül Üniversitesi Yönetim Bilişim Sistemleri bölümü Nesne Tabanlı Programlama dersi boyunca oluşturulan haftalık notları düzenli bir yapıda toplar.

## Klasör Yapısı

- `DersNotlari/`: Haftalara göre ayrılmış ders örnekleri


## İçerik

- `DersNotlari/Hafta-01` - `DersNotlari/Hafta-13`: Haftalık konu ve kod örnekleri
- `OzelSayilar`: Özel sayı kontrolleri ve örnek algoritmalar

## Çalıştırma

1. `Nesne Tabanlı Programlama.sln` dosyasını Visual Studio ile açın.
2. İstediğiniz haftanın veya projenin csproj dosyasını seçin.
3. Projeyi `Build` veya `Run` ile çalıştırın.

## Not

- Bazı projelerde `net7.0` kullanılıyor. Derleme sırasında destek süresi ile ilgili uyarı görülebilir.
